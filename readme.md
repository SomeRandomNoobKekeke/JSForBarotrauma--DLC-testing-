Experimental C# mod that adds JS Support for Barotrauma
Works only in DLC testing version and only on windows for now
Based on this template https://github.com/FakeFishGames/BaseBaroPlugin

Workshop page: https://steamcommunity.com/sharedfiles/filedetails/?id=3693078589

It uses https://github.com/ClearFoundry/ClearScript for js interpreting

Docs are here https://github.com/SomeRandomNoobKekeke/JSForBarotrauma--DLC-testing-/wiki
